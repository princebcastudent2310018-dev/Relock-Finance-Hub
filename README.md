# 💰 Relock Finance Hub

A modern web-based finance management application to track income, expenses, and generate detailed financial reports.

## 🚀 Features

- 📊 Monthly Summary (Income, Expenses, Savings, Net Worth)
- 📜 Transaction History with categories and dates
- 📈 Investment Report with portfolio insights
- ⬇️ Download reports in CSV format
- ⚡ Fast and user-friendly interface

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript / TypeScript
- Backend: Node.js (if used)
- Platform: Replit
